# スマート支払い判定

レジ前で「どのお金を出せばいいか」を瞬時に計算するWebアプリです。

財布の中の紙幣・硬貨の枚数を入力するだけで、支払い後の手持ちが最も少なくなる最適な組み合わせを自動で提案。2000円札にも対応しています。値引き券・パーセント割引クーポンも複数枚まとめて考慮した計算が可能です。

スマートフォンではテンキーUIで快適に入力でき、PCではキーボード入力にも対応。インストール不要、ブラウザだけで動作します。

## 機能

- 紙幣・硬貨の最適な組み合わせを計算（2000円札対応）
- 値引き券・パーセント割引クーポンの複合計算
- 支払い後の枚数最小化 / 小銭優先の2モード
- スマホ向けテンキーUI・PC向けキーボード入力を自動切り替え
- インストール不要・サーバー不要（HTMLファイル単体で動作）

## 使い方

1. 支払額を入力
2. 財布の中の枚数を入力
3. クーポンがあれば入力
4. 「計算する」を押す

## 動作環境

モダンブラウザ（Chrome / Safari / Firefox / Edge）
